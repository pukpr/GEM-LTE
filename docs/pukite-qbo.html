The basis uses a 2.368-year aliased draconic term plus a 173.31-day modulation envelope and added harmonics to emulate the formulation described by Pukite. It is an interactive approximation rather than a verbatim implementation of any unpublished source code.

Data and model notes

Observed data are NOAA PSL monthly 50 mb QBO values filtered to 1953-01 through 2024-12. The model framing follows Pukite's GeoEnergy Math posts: an aliased draconic cycle with modulation tied to the 173.3-day node-crossing interval and seasonal impulse structure.
